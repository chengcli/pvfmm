Construct a piecewise Chebyshev volume discretization in [0,1]^3. It first constructs a uniform tree of depth init_depth and then adaptively refines each leaf node until the tails of the Chebyshev coefficients in each leaf node are smaller than tol and the number of target points in each leaf node are less than max_pts. It further refines the tree to satisfy the 2:1 balance constraint.

Parameters
[in]cheb_degthe degree of the Chebyshev polynomials. The number of coefficients in each leaf node is data_dim*(cheb_deg+1)(cheb_deg+2)(cheb_deg+3)/6.
[in]data_dimthe number of scalar values per point in the evaluation of the input function pointer (fn_ptr).
[in]fn_ptrthe input function pointer.
[in]fn_ctxa context pointer to be passed to fn_ptr.
[in]trg_coordthe target coordinate vector with values: [x1 y1 z1 ... xn yn zn] where (x1 y1 z1) are the coordinates of the first target point.
[in]n_trgnumber of target points.
[in]commMPI communicator.
[in]tolthe tolerance for adaptive refinement.
[in]max_ptsthe maximum number of target points per leaf node.
[in]periodicwhether to use periodic boundary conditions.
[in]init_depththe depth of the initial tree before adaptive refinement. If zero then the depth is the minimum depth so that the number of leaf nodes is greater than the size of the MPI communicator.
Returns
the pointer to the constructed tree. It must be destroyed using PVFMMDestroyVolumeTreeD to free the resources.

Construct a piecewise Chebyshev volume discretization in [0,1]^3. It first constructs a tree with the given leaf node coordinates and then adds the Chebyshev coefficient to each leaf node.

Parameters
[in]Nleafthe number of leaf nodes.
[in]cheb_degthe degree of the Chebyshev polynomials. The number of coefficients in each leaf node is data_dim*(cheb_deg+1)(cheb_deg+2)(cheb_deg+3)/6.
[in]data_dimthe number of scalar values per point in the input density.
[in]leaf_coordA vector of points [x1 y1 z1 ... xn yn zn] where each point corresponds to a leaf node in the tree.
[in]fn_coeffthe vector of Chebyshev coefficients of size Nleaf*data_dim*(cheb_deg+1)(cheb_deg+2)(cheb_deg+3)/6, where Nleaf=leaf_coord.size()/3 is the number of leaf nodes.
[in]trg_coordthe target coordinate vector with values: [x1 y1 z1 ... xn yn zn] where (x1 y1 z1) are the coordinates of the first target point.
[in]n_trgnumber of target points.
[in]commMPI communicator.
[in]periodicwhether to use periodic boundary conditions.
Returns
the pointer to the constructed tree. It must be destroyed using PVFMMDestroyVolumeTreeD to free the resources.
